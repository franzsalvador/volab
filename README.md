# volab
A full stack project using MERN to build a social network that brings music producers and vocalists together to collaborate.

https://volab-app.herokuapp.com/

## Technologies Used
- Javascript
- HTML5
- CSS3
- Bootstrap
- React.JS
- MongoDB
- Express.JS
- Node.JS
